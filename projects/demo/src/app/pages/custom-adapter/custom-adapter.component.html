Custom DateTimeAdapter

Demonstrates how to implement a custom adapter and register it via DI. This LoggingDateTimeAdapter delegates all calls to the native adapter and logs parse/format operations to the browser console.

Implementing your own adapter

@Injectable()
class MyLuxonDateTimeAdapter extends NgxDateTimeAdapter<DateTime> {{ '{' }}
  // ... Luxon-based implementation ...
{{ '}' }}

// Global registration:
bootstrapApplication(AppComponent, {{ '{' }}
  providers: [
    provideNgxDatetimeKit({{ '{' }}
      adapter: MyLuxonDateTimeAdapter
    {{ '}' }})
  ]
{{ '}' }});

Full documentation at docs/adapter.md.
