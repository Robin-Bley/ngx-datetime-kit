Angular Material Integration

Import NgxMatFormFieldDirective to use any picker inside <mat-form-field>. The directive applies automatically via element selector — no extra attribute needed.

Basic usage

Every picker automatically adapts to Angular Material's visual container. The floating label and error messages are handled by mat-form-field.

Date (required) Please select a date. Date & Time Optional — no validation applied
@if (basicForm.valid && basicForm.dirty) {
Value: {{ basicForm.value | json }}
}

Range & DateTime Range pickers

The wider range pickers also work inside mat-form-field. The picker panel opens as usual — mat-form-field only wraps the trigger field.

Setup

Three steps to enable Angular Material integration in your application:

  1. Install peer dependency: npm install @angular/material
  2. Import the directive alongside your picker components and Angular Material form-field: 
    import {
  NgxDatePickerComponent,
  NgxMatFormFieldDirective,
} from 'ngx-datetime-kit';
import { MatFormFieldModule } from '@angular/material/form-field';

@Component({
  imports: [
    MatFormFieldModule,
    NgxDatePickerComponent,
    NgxMatFormFieldDirective, // ← activates automatically
  ],
})
  3. Use inside a form field: 
    <mat-form-field appearance="outline">
  <mat-label>Booking date</mat-label>
  <ngx-date-picker [formControl]="dateCtrl"></ngx-date-picker>
  <mat-error>Required</mat-error>
</mat-form-field>
